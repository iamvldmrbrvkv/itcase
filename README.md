# Каталог товаров - Тестовое задание

## Краткое описание

Интернет-магазин одежды с полным функционалом каталога и корзины.

## Основные возможности

- Каталог товаров с изображениями и ценами
- Выбор цвета товара
- Выбор размера с валидацией доступности
- Галерея изображений
- Корзина с уникальными товарами
- Сохранение корзины в localStorage
- Роутинг между страницами

## Старт

```bash
npm install
npm run dev
```

Откройте http://localhost:5173

## Структура проекта

```
src/
├── components/       # Header с навигацией
├── context/         # CartContext для управления корзиной
├── pages/           # ProductList, ProductDetail, Cart
├── services/        # API для получения данных
└── App.jsx          # Роутинг
```

## Фунционал

- Список товаров с изображениями
- Детальная страница товара
- Выбор цвета и размера
- Просмотр изображений
- Корзина с уникальными товарами
- Удаление товаров из корзины
- Счетчик товаров в корзине
- Кнопка корзины на всех страницах
- Сохранение в localStorage

## Технологии

- React 19
- React Router DOM
- Vite
- CSS
